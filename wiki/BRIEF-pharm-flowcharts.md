## 1. Жёсткие правила содержимого

### 1.1 Формуляр
- ВСЕГДА доступны: галоперидол, хлорпромазин (аминазин), левомепромазин (тизерцин), диазепам (сибазон), амитриптилин, клозапин, циклодол, бипериден, карбамазепин, литий.
- МОГУТ отсутствовать: оланзапин в/м, лоразепам, арипипразол в/м, кветиапин, ламотриджин, вортиоксетин, луразидон, эскетамин, дексмедетомидин.
- Для каждой рекомендации, в которой используется «может-отсутствовать»-препарат, ОБЯЗАТЕЛЬНО давать альтернативу из ВСЕГДА-доступных. Маркер: блок `АЛЬТЕРНАТИВА (если препарат недоступен):` внутри content.

### 1.2 Дозы
- Только конкретные: мг, путь, интервал, темп повторения/титрации.
- Источник для каждой дозы — из списка в §1.5. Если в источниках расхождение более 25% — давай оба значения с пометкой источника, не выбирай молча.
- Для титрации указывай день/неделю и целевую дозу, не просто «постепенно».

### 1.3 Диагностический язык
- Базовая система — МКБ-10. Где применимо — параллельная DSM-5 рубрика в скобках.
- Для феноменологии — постсоветская/снежневская номенклатура (паранояльный, парафренный, апатико-абулический, и т.д.) — для статуса и описания. Современная западная — для дифдиагностики и выбора эвиденс-терапии.

### 1.4 Стиль
- Без disclaimers «обратитесь к специалисту», «я ИИ» — это инструмент для практикующего психиатра.
- Не упрощать терминологию. Уровень детализации — Maudsley / Stahl, не санпросвет.
- Не вставлять рамочные обороты «важно отметить», «следует помнить», em-dash overuse, рул-оф-три, «не просто X, но Y». Клинически сухо.
- Латинские названия и МКБ-коды не переводить.

### 1.5 Допустимые источники
В порядке предпочтения:
1. Maudsley Prescribing Guidelines in Psychiatry (последнее доступное издание)
2. Stahl's Essential Psychopharmacology + Prescriber's Guide
3. Schatzberg & Nemeroff, Textbook of Psychopharmacology
4. NEI / Master Psychopharmacology (для рецепторного обоснования)
5. Bazire (Psychotropic Drug Directory)
6. Профильные guidelines: APA, NICE, CINP, WFSBP, CANMAT — для конкретных протоколов
7. Воронков, Дрейер, CHPD, Laux — для совместимости с уже существующими sources в репо
8. Для украинских реалий: наказы МОЗ, ЗУ «Про психіатричну допомогу», протоколы для экспертизы

ЗАПРЕЩЕНО как первичные источники: UpToDate-абстракты без указания первичного исследования, Wikipedia, медиа.

Для каждого flowchart'а в поле `sources` — массив строк формата `Author Year, edition, page/chapter` или ссылка на guideline.

### 1.6 Спорные темы
В этих темах требуется явный позиционный обзор pro/contra внутри content (не одностороннее «нельзя» или «можно»):
- бензодиазепины при ПТСР
- АД (особенно ТЦА и венлафаксин в высоких дозах) при БАР-2
- трамадол при опиоидной абстиненции и серотониновый риск
- прегабалин при тревоге vs misuse potential
- z-drugs при хронической инсомнии

## 2. Структура данных

Тип записи: `"type": "flowchart"`. Полная схема — как у `flowchart_ad` в `database.js`. Минимальный шаблон:

```json
{
  "id": "alg_<short_name>",
  "cmd_alias": "<short_name>",
  "type": "flowchart",
  "category": "<один из существующих>",
  "title": "<короткое название>",
  "tags": ["..."],
  "related": ["<id других записей>"],
  "description": "<1–2 предложения, что делает алгоритм>",
  "flowchart": {
    "start": "step1",
    "steps": {
      "step1": {
        "question": "...",
        "info": "<опционально, разъяснение перед ответом>",
        "options": [
          {"label": "...", "next": "step2"},
          {"label": "...", "next": "result_x"}
        ]
      },
      "result_x": {
        "result": true,
        "title": "...",
        "content": "ПЕРВАЯ ЛИНИЯ:\n• ...\n\nАЛЬТЕРНАТИВА (если препарат недоступен):\n• ...\n\nМОНИТОРИНГ:\n• ...\n\n⚠ КРАСНЫЕ ФЛАГИ:\n• ..."
      }
    }
  },
  "lastUpdated": "<YYYY-MM-DD>",
  "sources": [
    "Maudsley 14th ed., ch. X",
    "Stahl Prescriber's Guide 7th ed., entry on Y"
  ]
}
```

Категории — используй уже существующие (не создавай новые без необходимости). Существующие: «Ургентная психиатрия», «Аффективные расстройства», «Тревожные расстройства», «Наркология», «Военная психиатрия», «Расстройства сна», «Фармакотерапия (АД)», «Фармакотерапия (АП)», «Фармакотерапия (Нормотимики)», «Фармакотерапия (Общее)», и др.

Поле `related` — обязательно прокинь связи к минимум 2 уже существующим записям (по ids).

## 3. Скоуп задачи: 5 flowchart'ов

В порядке приоритета. Один flowchart — один коммит. После каждого — запусти build, проверь terminal1.html визуально (или хотя бы syntactic OK), коммить.

### 3.1 `alg_acute_agitation` — Острое психомоторное возбуждение
Категория: «Ургентная психиатрия». 
Существующая запись `alg_agitation` (matrix) — НЕ удаляй, она полезна как matrix-обзор. Создаём именно flowchart-wizard рядом, related связан с `alg_agitation`.

Развилки:
1. **Контактен / не контактен** (есть ли минимальная вербальная коммуникация и согласие на оральный приём)
2. **Этиология**: психотическое (шизо/мания) / алкогольное (тремор, вегетативная буря, запах) / делириозное (помрачение, флуктуация, «лёгкие сумерки») / тревожно-паническое (ясное сознание, без бреда) / органическое-неизвестное
3. **Доступ**: в/м (срочно, неконтактен) / в/в (только при наличии венозной линии и мониторинга) / п/о (контактен, согласен)
4. Внутри каждой ветки — препарат, доза, повтор-интервал, потолок суточной дозы

Обязательно:
- блок мониторинга (АД, ЧСС, ЧДД, SpO2, ЭКГ при кумулятивной дозе галоперидола >20 мг/сут)
- red flags (ригидность+лихорадка → ЗНС-stop; неэффективность на адекватных дозах → пересмотреть этиологию: гипогликемия, ЧМТ, интоксикация)
- предупреждение про комбинацию оланзапин в/м + БД в/м (избегать в первые 2 ч, риск кардиореспираторной депрессии)
- для алкогольного — отдельная ветка на CIWA-Ar и переход к `alg_alcohol_withdrawal` (см. 3.2)

Источники: Maudsley (chapter on acute behavioural disturbance), NICE NG10, Stahl PG (haloperidol, diazepam, chlorpromazine entries), BAP guidelines on acute behavioural disturbance.

### 3.2 `alg_alcohol_withdrawal` — Алкогольная абстиненция: стратификация и фарма
Категория: «Наркология».

Развилки:
1. **CIWA-Ar score** (укажи, что калькулятор есть в database — `related` к нему): <10 (лёгкая) / 10–18 (средняя) / >18 (тяжёлая) / признаки осложнённого синдрома (судороги, делирий tremens — отдельная ветка)
2. Внутри тяжести — выбор протокола: front-loaded vs symptom-triggered
3. **Front-loaded** (тяжёлая, риск ДТ): диазепам 10–20 мг п/о или в/в каждый час до седации/CIWA <10
4. **Symptom-triggered** (средняя): диазепам 10 мг при CIWA ≥10, повтор каждый час
5. **Лёгкая**: тиамин + поддержка, наблюдение
6. Параллельно во всех ветках: тиамин 200–500 мг в/м/в/в ДО глюкозы, магний при дефиците, регидратация
7. Red flags для перевода в реанимацию: рефрактерность к 80+ мг диазепама за 4 ч, дыхательная депрессия, эпистатус, кома.

Не предлагай хлордиазепоксид, лоразепам, оксазепам без альтернативы — диазепам в Украине стандарт. Если упоминаешь — обязательно с пометкой о доступности.

Источники: Maudsley (alcohol withdrawal), BAP guidelines on alcohol misuse, NICE CG100, ASAM National Practice Guideline.

### 3.3 `alg_bipolar_depression` — Биполярная депрессия: выбор фармы
Категория: «Аффективные расстройства».

Это место постоянных ошибок. Алгоритм должен явно блокировать назначение СИОЗС-монотерапии при БАР.

Развилки:
1. БАР-1 vs БАР-2 (с напоминанием о критериях гипомании DSM-5/МКБ-10)
2. Острая депрессия vs поддерживающая фаза
3. Доступ к препаратам: «есть кветиапин» / «есть ламотриджин» / «есть только базовый набор»
4. Текущая фарма: на стабилизаторе vs нет
5. Тяжесть: с психотическими симптомами / без / суицидальная

Резисты: переходы на литий, ламотриджин (с титрацией под контролем сыпи — отдельный мини-блок про SJS), кветиапин, lurasidone (если есть). ЭСТ при суицидальном БАР-1.

Если предлагаешь АД — обязательно: только под прикрытием стабилизатора, предпочтение бупропион/СИОЗС, избегать ТЦА и венлафаксина ≥150 мг (риск induction). Это явно прописать в каждом result, где упоминаются АД.

Источники: CANMAT 2018 bipolar guidelines, BAP 2016 bipolar, Maudsley (bipolar chapter), Stahl PG (lamotrigine, lithium, quetiapine, lurasidone entries).

### 3.4 `alg_clozapine_initiation` — Клозапин: стартап и мониторинг
Категория: «Фармакотерапия (АП)».

Это не классический wizard, а time-axis протокол. Реализуй как flowchart с шагами по «фазам», где каждый узел — это фаза с конкретными задачами:

1. **Pre-start**: лейкоформула, ЭКГ, тропонин (baseline), CRP, глюкоза, липиды, вес, исключение миокардита-факторов риска
2. **День 1–14 (титрация)**: 12.5 → 25 → 50 → 100 → 150 → 200 мг с темпом, ортостаз контроль, тахикардия, седация
3. **Неделя 1–4 (миокардит-окно)**: тропонин и CRP еженедельно, симптомы (одышка, тахикардия в покое >100, лихорадка, грудная боль) → STOP и кардиолог
4. **Неделя 1–18 (агранулоцитоз-окно)**: лейкоформула еженедельно, пороги остановки (нейтрофилы <1.5 → пауза, <1.0 → отмена, <0.5 → reverse isolation)
5. **Неделя 18 — год**: лейкоформула каждые 2 недели
6. **После года**: лейкоформула ежемесячно
7. **Долгосрочно**: метаболический мониторинг, гиперсаливация, констипация (риск илеуса, мониторинг стула обязателен — это причина смерти)

Каждая фаза — отдельный result-узел с детализацией. Связи между ними через question-узлы вида «На какой фазе пациент?».

Источники: Maudsley (clozapine chapter, обширно), Stahl PG (clozapine), CINP guidelines on clozapine, специфические работы по миокардиту (Ronaldson 2015, Citrome 2016).

### 3.5 `alg_combat_ptsd_circuits` — ПТСР боевой: фарма по нейроконтурам
Категория: «Военная психиатрия».

Используй существующую матрицу нейроконтуров (`combat_trauma_circuit_matrix.html`, `combat_trauma_syndromokinesis.html`). Прочитай оба файла перед началом, согласуй терминологию контуров: страха, интрузий, диссоциации, морального ранения, гипервозбуждения, оцепенения.

Развилки:
1. **Доминирующий контур** (можно несколько, тогда матричный выбор)
2. Для каждого — фарма-опции:
   - Страх + гипервозбуждение → СИОЗС (сертралин, пароксетин — единственные с FDA-approval для ПТСР) / венлафаксин ER; ± празозин для ночных кошмаров (если доступен), ± пропранолол on-demand при приступах
   - Интрузии → СИОЗС + потенциация атипичным AP в низких дозах (рисперидон 0.5–2 мг, кветиапин 25–100 мг — если доступен)
   - Диссоциация → ОСТОРОЖНО с БД (риск углубления), низкие дозы AP, рассмотреть налтрексон при тяжёлой
   - Моральная рана → фарма ограничена; основная работа — экзистенциальная терапия (ссылка на психотерапевтический модуль если он есть)
   - Оцепенение/анергия → активирующие АД (бупропион, СИОЗСН в адекватных дозах), исключение коморбидной депрессии и ЧМТ
3. Особые подситуации: алкогольный ко-юз (отдельная ветка — детокс прежде ПТСР-фармы), суицидальность, военно-врачебная экспертиза (упомянуть, что фарма-history влияет на ВВК)
4. Для острой и подострой фазы — ссылка на острый stress-протокол (если есть в базе), не путать с хронической ПТСР

Спорные позиции, расписать pro/contra:
- БД при ПТСР: ASAM против, но on-demand при гипервозбуждении некоторые pratiques используют; явно описать риски (диссоциация, зависимость, аннулирование экспозиционной терапии)
- Каннабис/CBD: набирающая популярность тема среди ветеранов, не игнорировать, но критически

Источники: VA/DoD Clinical Practice Guideline for PTSD (latest), ISTSS 2018, APA PTSD guideline 2017, специфические работы по prazosin (Raskind 2018 — отрицательный mainline trial, но subgroup-эффекты), литература по moral injury (Litz 2009).

## 5. Что делать при сомнениях

- Расхождение в дозах между источниками >25% → в content флаг `[CONFLICT: src A — N мг, src B — M мг]`, не выбирать молча.
- Препарат не упомянут ни в ВСЕГДА, ни в МОГУТ отсутствовать → не использовать, либо вынести в отдельный комментарий PR с вопросом, добавить ли его.
- Нужна категория, которой нет → не создавай новую без явного разрешения, используй ближайшую существующую и пометь в PR.
- Вопросы по украинским юридическим нюансам (учёт, ВВК) → не выдумывать, флаг в PR.

## 6. Out of scope

Не делай в этой ветке:
- Изменения в `terminal1.html` (frontend) — только данные.
- Новые типы записей кроме `flowchart`.
- Психотерапевтические модули — только фарма.
- Перевод существующих matrix/table в flowchart — это отдельный refactor.
